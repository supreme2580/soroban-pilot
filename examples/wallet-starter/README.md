# Wallet Starter

A starter wallet application demonstrating Stellar wallet integration.

## Features

- Wallet connection (Freighter, xBull, Albedo, Rabet)
- Balance display
- Transaction history
- Send and receive payments

## Setup

```bash
pnpm install
pnpm dev
```

## Architecture

- React + TypeScript frontend
- Stellar SDK for blockchain interaction
- Wallet SDK for connection management
