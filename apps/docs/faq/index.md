# Frequently Asked Questions

## General

**What is StellarPilot?**
StellarPilot is an open-source toolkit for building applications on the Stellar Network.

**Do I need to know Rust?**
Only for Soroban smart contract development. The SDK and UI packages are TypeScript.

## Development

**What networks are supported?**
Testnet, Mainnet, and Standalone (local).

**Can I use any wallet?**
StellarPilot supports Freighter, xBull, Albedo, and Rabet.

## Templates

**Can I add my own template?**
Yes, templates are just directories. See the templates package for examples.
