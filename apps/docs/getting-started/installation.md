# Installation

## Prerequisites

- Node.js 18+
- pnpm (recommended) or npm
- Git

## Quick Start

```bash
npx @stellarpilot/cli create my-app
cd my-app
pnpm install
pnpm dev
```

## Manual Setup

```bash
mkdir my-stellar-app && cd my-stellar-app
pnpm init
pnpm add @stellarpilot/sdk
```

## Requirements

- Node.js >= 18
- pnpm >= 8
- For Soroban development: Rust and Soroban CLI
