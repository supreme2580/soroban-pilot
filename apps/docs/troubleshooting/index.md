# Troubleshooting

## Common Issues

### Wallet connection fails

Ensure the wallet extension is installed and unlocked.
Try refreshing the page and reconnecting.

### Transaction fails

- Check your account has sufficient balance
- Verify the network matches (testnet vs mainnet)
- Ensure your account is funded on the target network

### Build errors

```bash
pnpm install
pnpm build
```

Clear node_modules and reinstall if issues persist.
